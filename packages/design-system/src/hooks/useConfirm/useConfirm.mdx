# useConfirm

The `useConfirm` hook provides methods to manage confirmation dialogs. It is designed to be used within a `ConfirmProvider`.

## API

### `useConfirm`

This hook returns a function that can be used to trigger confirmation dialogs.

- **Type**: `() => (options: ConfirmOptions) => Promise<boolean>`
- **Parameters**:
  - `options`: Properties for the confirmation dialog.
    - **Type**: `ConfirmOptions`
    - **Properties**:
      - `title`: The title of the confirmation dialog.
        - **Type**: `string`
        - **Required**: Yes
      - `notice`: The notice message displayed in the dialog.
        - **Type**: `string`
        - **Optional**: Yes
      - `confirmLabel`: The label for the confirm button.
        - **Type**: `string`
        - **Optional**: Yes
        - **Default**: `'Yes'`
      - `cancelLabel`: The label for the cancel button.
        - **Type**: `string`
        - **Optional**: Yes
        - **Default**: `'No'`
      - `onConfirm`: Callback function to execute when confirmed.
        - **Type**: `() => void`
        - **Optional**: Yes
      - `onCancel`: Callback function to execute when canceled.
        - **Type**: `() => void`
        - **Optional**: Yes

## Example

Use the `useConfirm` hook to manage confirmation dialogs within your component.

```tsx
import React from 'react';
import { useConfirm } from '@siteed/design-system/hooks/useConfirm';
import { Button, View, Text } from 'react-native';

const App = () => {
  const confirm = useConfirm();

  const handleDelete = async () => {
    const result = await confirm({
      title: 'Delete Item',
      notice: 'Are you sure you want to delete this item?',
      confirmLabel: 'Yes',
      cancelLabel: 'No',
    });

    if (result) {
      console.log('Item deleted');
    } else {
      console.log('Delete canceled');
    }
  };

  return (
    <View>
      <Text>Hello World</Text>
      <Button title="Delete Item" onPress={handleDelete} />
    </View>
  );
};

export default App;
```

For more details on the ConfirmProvider, refer to the [ConfirmProvider](?path=/docs/providers--docs#confirmprovider) documentation.
