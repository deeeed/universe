# useModal

The `useModal` hook provides methods to manage custom modals, including bottom sheet modals and regular modals. It is designed to be used within a `CustomModalProvider`.

## API

### `dismiss`

Dismiss a specific modal.

- **Type**: `(key?: string) => boolean`
- **Parameters**:
  - `key`: Optional key to identify the modal to be dismissed.
- **Returns**: `boolean`

### `editProp`

Open a modal for editing a property.

- **Type**: `(props: EditPropProps) => Promise<DynInputProps['data']>`
- **Parameters**:
  - `props`: Properties for the modal and the data to be edited.
- **Returns**: `Promise<DynInputProps['data']>`

### `openDrawer`

Open a custom bottom sheet modal drawer.

- **Type**: `(props: OpenDrawerProps) => Promise<unknown>`
- **Parameters**:
  - `props`: Properties for the bottom sheet modal drawer.
- **Returns**: `Promise<unknown>`

### `openModal`

Open a custom modal.

- **Type**: `(props: OpenModalProps) => Promise<DynamicType>`
- **Parameters**:
  - `props`: Properties for the modal.
- **Returns**: `Promise<DynamicType>`

### `dismissAll`

Dismiss all open modals.

- **Type**: `() => void`

### `bottomSheetModalRef`

Ref object for the `BottomSheetModal`.

- **Type**: `React.RefObject<BottomSheetModal>`

## Example

Customize the modals by using the `openDrawer`, `openModal`, or `editProp` methods provided by the `useModal` hook.

```tsx
import React from 'react';
import { useModal } from '@siteed/design-system/hooks/useModal';
import { Button, View, Text } from 'react-native';

const App = () => {
  const { openDrawer, openModal } = useModal();

  const handleOpenDrawer = async () => {
    const result = await openDrawer({
      title: 'Custom Drawer',
      footerType: 'confirm_cancel',
      initialData: { name: 'John Doe' },
      render: ({ resolve, reject }) => (
        <View>
          <Text>Drawer Content</Text>
          <Button title="Confirm" onPress={() => resolve('Confirmed')} />
          <Button title="Cancel" onPress={() => reject('Canceled')} />
        </View>
      ),
    });

    console.log('Drawer Result:', result);
  };

  const handleOpenModal = async () => {
    const result = await openModal({
      initialData: { age: 30 },
      render: ({ resolve, reject }) => (
        <View>
          <Text>Modal Content</Text>
          <Button title="Confirm" onPress={() => resolve({ age: 31 })} />
          <Button title="Cancel" onPress={() => reject(new Error('Canceled'))} />
        </View>
      ),
    });

    console.log('Modal Result:', result);
  };

  return (
    <View>
      <Text>Hello World</Text>
      <Button title="Open Drawer" onPress={handleOpenDrawer} />
      <Button title="Open Modal" onPress={handleOpenModal} />
    </View>
  );
};

export default App;
```

For more details on the CustomModalProvider, refer to the [CustomModalProvider](?path=/docs/providers--docs#custommodal) documentation.
