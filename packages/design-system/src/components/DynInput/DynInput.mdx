# DynInput

`DynInput` is a versatile input handler for React applications, supporting various input types like text, textarea, number, radio, select-button, and custom render types. It provides a flexible interface for handling different forms of user input.

## Important Usage Note

The `DynInput` component is not intended to be used as a standalone component. It is designed to be used in conjunction with the `editProp` function from `useBottomModal` to quickly edit a value within a bottom sheet modal context.

## Usage with useBottomModal

To use `DynInput` properly, you should utilize it with the `editProp` function from `useBottomModal`. Here's a basic example:

```jsx
import { useBottomModal } from '../path/to/useBottomModal';
import { DynInput } from '../path/to/DynInput';

function MyComponent() {
  const { editProp } = useBottomModal();

  const handleEdit = async () => {
    const result = await editProp({
      data: "Initial value",
      inputType: "text",
      // other DynInput props as needed
    });
    console.log("Edited value:", result);
  };

  return (
    <Button onPress={handleEdit}>
      Edit Value
    </Button>
  );
}
```

In this setup, `DynInput` is rendered within a bottom sheet modal, providing a smooth editing experience for the user.

## Examples

While `DynInput` is not meant to be used standalone, here are some examples of how it might appear within the bottom sheet modal:

### Text Input

```jsx
<DynInput data="Hello World" inputType="text" />
```

### Number Input

```jsx
<DynInput data={42} inputType="number" />
```

### Select Button

```jsx
<DynInput 
  data={[
    { label: "Option 1", value: 1, selected: false },
    { label: "Option 2", value: 2, selected: true },
    { label: "Option 3", value: 3, selected: false }
  ]} 
  inputType="select-button" 
/>
```

Remember, these examples demonstrate the appearance of `DynInput` within the bottom sheet modal, not how you would use the component directly in your JSX.
