### Breaking Changes Addressed
1. **newArchEnabled removed**: SDK 55 only supports New Architecture; removed `newArchEnabled: true` from designdemo's `app.config.ts`
2. **Expo SDK package versioning**: All Expo packages now use major version 55 (e.g., expo-router ~55.0.2)
3. **RN 0.83 type changes**: Fixed `RefreshControl.size` type (now `'default' | 'large'` instead of number) and `TextInput` focus/blur event types (now `FocusEvent`/`BlurEvent` instead of `NativeSyntheticEvent<TextInputFocusEventData>`)
4. **tsconfig moduleResolution**: Updated design-system from `"node"` to `"bundler"` to match expo/tsconfig.base's `customConditions`
